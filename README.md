# Restaurant Management System

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## Features

### Admin Panel
- Register & Login
- Manage Menu (Add, Edit, Update, Delete)
- View Customer Details
- Approve Orders
- Update Delivery Status

### User Panel
- Register & Login
- Order Items
- View Order Status
- View Order History

## Setup Instructions
1. Clone the repository
2. Import the database.sql file into phpMyAdmin
3. Run using XAMPP (Apache + MySQL)


## Admin Panel Access

The admin authentication pages are not exposed on the main navigation for security purposes.

Access them directly via:

- /admin_register.php
- /admin_login.php

Example:
http://localhost/restaurant-management-system/admin_login.php
